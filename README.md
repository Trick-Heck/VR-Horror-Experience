# VR Horror Experience (Unity)

Experiencia de terror psicológico desarrollada en realidad virtual utilizando Unity.

El proyecto busca generar tensión en el jugador mediante el uso de iluminación ambiental, diseño de escenario inmersivo y audio espacial.

## Mi rol en el proyecto

Diseñé el escenario principal de la experiencia VR utilizando assets modulares dentro de Unity.

Implementé e integré el sistema de audio ambiental y efectos sonoros espaciales para reforzar la inmersión del jugador.

Colaboré en la construcción de la atmósfera narrativa mediante iluminación y ambientación del entorno.

Proyecto desarrollado en colaboración académica.

## Tecnologías utilizadas

Unity
C#
XR Interaction Toolkit
Spatial Audio
Lighting Design
